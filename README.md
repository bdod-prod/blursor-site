# blursor-site

Source for blursor.ai — research distillation site for AI visibility, LLM
ranking factors, and how generative engines decide what sources to cite.

## Deployment

Push to `main` triggers a GitHub Action that deploys to Cloudflare Pages
(project: `blursor`). Custom domain: blursor.ai.

## Content

Articles under `research/` are mostly auto-generated by the BLURSOR pipeline
(n8n + Claude). Auto-generated pull requests should be reviewed for voice and
factual accuracy before merge.

## Structure

```
index.html              ← homepage
research/index.html     ← article archive
research/<slug>.html    ← individual articles (one per arXiv paper)
author/                 ← author bio pages
references/             ← internal design references (not linked from nav)
images/                 ← logos and sample paper screenshots
content_md/             ← markdown drafts (kept in repo for history)
screencaps/             ← deploy state evidence
assets/                 ← OG images and other shared assets
```
